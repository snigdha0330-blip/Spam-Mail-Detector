# Spam Mail Detector

This project classifies SMS messages as Spam or Ham using Machine Learning.

## Technologies Used
- Python
- Scikit-learn
- TF-IDF Vectorizer
- Multinomial Naive Bayes

## Accuracy
96%

## Dataset
SMS Spam Collection Dataset
